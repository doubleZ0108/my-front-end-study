Canvas 画布

画布是矩形区域，可以控制每一个像素

canvas元素本身没有绘图能力，所有绘图工作通过JavaScript实现

➡️为x轴正向
⬇️为y轴正向

矩形

cxt.fillRect(5,20,175,50); 起始点为(5,20)
矩形的 长为175 宽为50

线🧵

cxt.lineTo(300,150); 从cxt当前位置做直线到（300，150）

圆

cxt.arc(200,100, 40, 0,2*Math.PI, true); 圆心坐标（200，100）
半径 40
起始角 0 * Math.PI
结束角 2 * Math*PI
逆时针 true

渐变

let grd = cxt.createLinearGradient(20,40, 300,150); grd.addColorStop(0,'#e72a23'); grd.addColorStop(1,'#fff063'); cxt.fillStyle = grd; 从cxt当前位置做直线到（300，150）

图像

cxt.lineTo(300,150); 从cxt当前位置做直线到（300，150）
