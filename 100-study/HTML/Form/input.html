表单

用于搜集不同类型的用户输入

表单元素包括 input | 复选框 | 单选按钮 | 提交按钮

input

text 文本输入

单行输入，文字字段的默认宽度是20个字符

请输入昵称：

请输入学校：

password 密码

请输入密码：

file 文件

multiple属性允许用户上传多个文件

请选择文件：

number 数字字段

请输入(选择)重量：1～5之间

email 邮箱📮

输入错误提示：请在电子邮箱地址中包括"@"。"xxx"中缺少"@"。

请输入邮箱：

data 日期📅

请选择日期：

color 颜色

请输入颜色：

datalist 为input元素规定预定义选项列表

用户在输入数据时可以看到预定义选项的下拉列表

input元素的list属性必须引用datalist元素的id属性

radio 单选按钮

同name的radio互斥 | checked指定默认选项

请选择性别：
male
female

checkbox 复选框

允许用户选择0个或多个

请选择考试科目：
语文
数学
英语

submit 提交按钮

向 表单处理程序 提交表单的按钮

action属性中指定表单处理程序，会向这个程序传递表单的数据

method指定提交表单时所用的HTTP方法: 使用get时表单数据在地址栏中可见,
每个输入字段必须设置一个name属性，才能正确的提交，如果某个input没有name，则它不会被提交
