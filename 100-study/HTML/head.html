title -> 指定文档的标题

<title>HTML 头部head 学习</title>
  • 定义浏览器工具栏中的标题
  • 提供页面被收藏到收藏夹时显示的标题
  • 显示在搜索引擎结果中的页面标题

link -> 引用外部样式表

<link rel="stylesheet" type="text/css" href="static/css/style.css" />

style -> 在HTML文档中定义样式

<style> h1{ color: rebeccapurple; } </style>

script -> 定义客户端脚本

<script src="static/js/common.js" type="text/javascript"></script> <script> document.write("<h1>head中标签的使用</h1>\n" + "<hr />") </script>

base -> 使所有链接指向同一链接🔗

<base target="_blank" />

通过base标签可以使所有链接指向指定界面 | 在新窗口打开

我没自己指定在新窗口打开，但是我会在新窗口显示

meta -> 元数据 & 描述文档

标识了创作者和编辑软件

<meta name="author" content="https://doubleZ.com" /> <meta name="revised" content="double Z, 08/06/2019" /> <meta name="generator" content="WebStorm 2019.2" />

文档描述和关键词

<meta name="description" content="study the header of HTML" /> <meta name="keywords" content="header, title, base, meta" />

重定向用户

<meta http-equiv="Refresh" content="5;url=https://github.com/doubleZ0108/" />
