border 边框

将所有属性写在一个属性中

p{ border: red solid 1px; }

border-style 边框的样式

dotted 点状

dashed 虚线

solid 实线

double 双线

groove 3D凹槽

ridge 3D垄状

inset 3Dinset

outset 3Doutset

inherit 从父元素继承边框样式

每一边设置不同的边框

p.mix4{ border-style: dotted solid double dashed; } p.mix4{ border-top-style: dotted; border-right-style: solid; border-bottom-style: double; border-left-style: dashed; }

mix4「dotted solid double dashed」

mix3「dashed solid double」

mix2「dashed solid」

可以把图片链接边框设置为outset，使之看上去像是凸起按钮

a:link img{ border-style: outset; }

border-width 边框的宽度

p.thick{ border-style: solid; border-width: thick; } p.mix-width{ border-style: solid; border-width: 15px 5px 20px 50px; }

thin

thick

mix-width「15px 5px 20px 50px」

border-color 边框的颜色

mix-color「blue rgb(25%,35%,45%) #909090 red」

This is a transparent link.
