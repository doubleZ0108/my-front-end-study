3D转换

  • transform-style 子元素是否保留3D转换 div{ transform-style: flat; /*子元素不保留其3D位置*/ transform-style: preserve-3d; /*子元素保留其3D位置*/ }
  • perspective 透视效果

    定义3D元素距试图的距离，以像素记

    当为元素定义透视属性时，其子元素会获得透视效果，而不是元素本身 div{ perspective: 500; /*子元素不保留其3D位置*/ }
  • backface-visibility 元素背对屏幕时是否可见 div{ backface-visibility: visible; /*背面是可见的*/ backface-visibility: visible; /*背面是不可见的*/ }
