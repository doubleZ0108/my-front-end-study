text-shadow 文本阴影

text-shadow: h-shadow v-shadow blur color;

  • h-shadow: 水平阴影的位置
  • v-shadow: 垂直阴影的位置
  • blur: 模糊的距离(可选)

  • 文本阴影效果

  • 带模糊的文本阴影效果

  • p.text-shadow-white{ color: white; text-shadow: 2px 2px 4px black; }

    白色文本的阴影效果

  • p.text-shadow-lamp{ color: red; text-shadow: 0 0 3px yellow; }

    霓虹灯效果的文本阴影效果
