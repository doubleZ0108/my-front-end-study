伪类

锚伪类

  • a:link{} 未访问的链接
  • a:hover{} 已访问的链接
  • a:visited{} 鼠标移动到链接傻姑娘
  • a:active{} 选定的链接(比如拖拽时)
  • ⚠️顺序必须是 link, visited >> hover >> active (>>表示早于)
