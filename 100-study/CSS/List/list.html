列表

将所有属性写在一个声明中

li{ list-style: url("xxx.jpg") square inside; }

list-style-type 列表类型

默认
  • disc
ul.circle{ list-style-type: circle; }
  • circle
ul.square{ list-style-type: square; }
  • square
大/小写罗马数字 ol.lower-roman{ list-style-type: lower-roman; }
  1. lower-roman
  2. lower-roman
  3. lower-roman
大/小写字母 ol.upper-alpha{ list-style-type: upper-alpha; }
  1. upper-alpha
  2. upper-alpha
  3. upper-alpha
小写希腊字母 ol.lower-greek{ list-style-type: lower-greek; }
  1. lower-greek
  2. lower-greek
  3. lower-greek

list-style-image 列表项图片

ul.ul-image li{ list-style-image: url("../../Resources/img/flower_icon.png"); }
  • 00001
  • 00002

list-style-position 列表标识位置

ul.inside{ list-style-position: inside; }
  • inside
  • inside
