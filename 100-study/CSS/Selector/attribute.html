属性和值选择器 - 多个值

  • 选择属性值中包含指定词汇的属性 [title~=hello]{ color: green; } <p title="hello world">hello world</p> <a title="欢迎你来到 hello" href="https://github.com/doubleZ0108/">欢迎你来到 hello</a>

    hello world

    欢迎你来到 hello
  • 选择属性值以指定开头的属性，盖茨必须是这个那个单词 [content|=en]{ color: coral; } <p content="en">en</p> <p content="en-our">en-our</p> <p content="en this">en this</p> <p content="us-en">us-en</p>

    en

    en-our

    en this

    us-en

  • 匹配属性值以指定值开头的元素 [dict^=man]{ color: deeppink; } <p dict="man">man</p> <p dict="man-this">man-this</p> <p dict="man this">man this</p>

    man

    man-this

    man this

  • 匹配属性值以指定值结尾的元素 [list$=boy]{ color: navy; } <p list="boy">boy</p> <p list="this-boy">this-boy</p> <p list="this boy">this boy</p>

    boy

    this-boy

    this boy

  • 匹配属性值包含指定值 [set*=child]{ color: yellowgreen; } <p set="this-child-that">this-child-that</p> <p set="this-child">this-child</p> <p set="child-that">child-that</p>

    this-child-that

    this-child

    child-that
