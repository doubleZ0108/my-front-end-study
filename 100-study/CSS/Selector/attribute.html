属性和值选择器 - 部分属性值选择

  • [=]: 必须完全匹配
  • [~=]: 选择属性值中包含指定词汇的属性 [title~=hello]{ color: green; } <p title="hello world">hello world</p> <a title="欢迎你来到 hello" href="https://github.com/doubleZ0108/">欢迎你来到 hello</a>

    hello world

    欢迎你来到 hello
  • [^=]: 匹配属性值以指定值开头的元素 [dict^=man]{ color: deeppink; }

    man

    man-this

    man this

  • [$=]: 匹配属性值以指定值结尾的元素 [list$=boy]{ color: navy; }

    boy

    this-boy

    this boy

  • [*=]: 匹配属性值包含指定值 [set*=child]{ color: yellowgreen; }

    this-child-that

    this-child

    child-that

  • [|=]: 选择属性值以en或en-开头的所有元素 [content|=en]{ color: coral; }

    en

    en-our

    en this

    us-en


    假设有很多图片，命名规则都是figure-1.jpg figure-2.png等等

    可以使用如下选择器匹配所有图像 img[src|=figure]{ ... }
