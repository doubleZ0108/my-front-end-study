类选择器

以 . 定义，类名不能以数字开头

*.center{ text-align: center; } 可以简写成 .center{ text-align: center; } <h6 class="center">This H6 will be center-aligned.</h6> <p class="center">This paragraph will also be center-aligned.</p>
This H6 will be center-aligned.

This paragraph will also be center-aligned.

类选择器也经常与派生选择器共同使用

类名为center的表格元素内部的表格单元的样式 .center td{ ... }

类选择器还常基于元素选择

只有类名为center的表格单元生效，类名为center的段落标签不受影响 td.center{ ... }

多类选择器

一个class值中可以包含一个词列表，每个词之间用空格分隔

p.important{ font-weight: bold; } p.warning{ color: red; } p.important.warning{ /*⚠️这里两个类没有空格*/ background-color: yellow; }

This is paragraph is important.

This is paragraph is warning.

This is paragraph is both important and warning.

This is paragraph is 「important & warning & urgent」.
