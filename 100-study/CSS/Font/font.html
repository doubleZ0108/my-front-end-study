font-family 使用字体系列

  • 使用通用字体系列 body{ font-family: sans-serif; }
  • 使用指定字体系列 h1{ font-family: Georgia, serif; }
  • 指定一系列类似的字体作为备选 p{ font-family: Times, "Times New Roman", "New Century Schoolbook", Georgia, 'New York', serif; }

font-size 字体大小

  • 绝对值
    • 将文本设置为指定的大小
    • 不允许用户改变文字大小
    • 绝对大小在确定了输出的物理尺寸时很有用
  • 相对值
    • 相对于周围元素来设置大小
    • 允许用户在浏览器中改变文字大小
  • 像素 p.pixel{ font-size: 16px; }

    16 pixel

  • em p.em-larger{ font-size: 2.5em; } p.em-smaller{ font-size: 0.6em; }

    2.5 em

    0.6 em

  • 百分比和em body{ font-size: 100%; } p{ font-size: 1.5em; }
