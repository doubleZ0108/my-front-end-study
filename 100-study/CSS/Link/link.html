链接的状态

  • a:link - 普通的、未被访问的链接
  • a:visited - 用户已访问的链接
  • a:hover - 鼠标位于链接上方
  • a:active - 链接被点击的时刻

text-decoration 去掉链接的下划线

a:link{ text-decoration: none; } a:hover{ text-decoration: underline; }

font-size 鼠标放上去的时候放大

a:hover{ font-size: 150%; }

background-color 鼠标放上去的时候有背景颜色

a:hover{ background-color: aquamarine; }
