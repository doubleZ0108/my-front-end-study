z-index 元素的堆叠顺序

设置一个定位元素沿z轴的位置，z轴为垂直延伸到显示区的轴。如果为正数，则离用户更近

  • 较高堆叠顺序的元素处在上面
  • 元素可拥有负的z-index属性值
    • 仅能在定位元素上奏效(例如position: absolute;)

图片衬于文字下

img.up{ position: absolute; z-index: -1; }
