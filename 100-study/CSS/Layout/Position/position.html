定位

position 定位类型

值 描述
absolute 绝对定位，相对于最近的以定位祖先元素(可以理解为非static定位的祖先)定位，如果没有则就是相对于body定位
fixed 固定定位，相对于浏览器窗口进行定位
relative 相对定位
static 默认值，没有定位(忽略top,bottom,left,right以及z-index声明)
